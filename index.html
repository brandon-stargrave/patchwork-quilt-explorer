How to Build a Formula
A formula is evaluated at every pixel using its x,y coordinates (and optionally t for animation).

1. Start with a combination — pick two variables or a pattern block and join them with an operator.
Example: click x then XOR ^ then y → x ^ y

2. Layer more operations — shift bits, mask with &, or square the result to add complexity.
Example: (x ^ y) >> 2 or (x ^ y) & 0xFF

3. Use parentheses — select part of your formula and click Wrap (…) to group it. This controls evaluation order and prevents syntax errors.

4. Add t for animation — expressions like (y - t) >> 3 create a scrolling effect. Hit Play below to animate.

Tips: • & 1 at the end gives crisp 1-bit patterns. • >> 4 zooms out the pattern. • Squaring (**2) then shifting (>> 12) creates Knuth-style fractals. • Try a Common Pattern as a starting point.
